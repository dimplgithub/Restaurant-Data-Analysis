# 🍽️ Restaurant Data Analysis and Rating Prediction using Python

## Cognifyz Technologies – Data Science Internship

This project was completed as part of the **Cognifyz Technologies Data Science Internship**. It focuses on analyzing restaurant data through data preprocessing, exploratory data analysis (EDA), feature engineering, predictive modeling, and data visualization to extract meaningful insights and predict restaurant ratings.

---

# 📌 Project Overview

The project demonstrates a complete end-to-end Data Science workflow using Python.

The analysis includes:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Geospatial Analysis
- Feature Engineering
- Customer Preference Analysis
- Predictive Modeling
- Data Visualization

The final machine learning model predicts restaurant aggregate ratings using **Random Forest Regression**.

# 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/dimplgithub/Cognifyz-Restaurant-Data-Analysis.git
```

### 2. Navigate to the project directory

```bash
cd Cognifyz-Restaurant-Data-Analysis
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Run the notebooks in order:

- Level1.ipynb
- level2.ipynb
- level3.ipynb
